Đếm tần suất các thuộc tính trong object
Viết hàm countProperties(obj) nhận vào một object và trả về một object mới chứa tần suất xuất hiện của các thuộc tính trong object đầu vào.
- Input:
+ obj: object đầu vào
- Output:
+ object mới chứa tần suất xuất hiện của các thuộc tính trong object đầu vào
- Ví dụ: + Input: {a: 1, b: 2, c: 1, d: 2} + Output: 4
